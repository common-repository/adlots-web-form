Adlots contact form plugin. 


== Description ==


Simple non-bloated WordPress Contact Form. The functionality of  Adlots Web-Form allows you to receive information that user fills in and, based on it, to automate the process of creating contacts  in Pipedrive using the plugin for WordPress;
View in real-time who is browsing your website and what pages they have visited.
For the complete list of features and details, Please visit <a rel="nofollow" href="http://doc.tenderp.com/display/AD/Adlots">Adlots documentation space</a> for more details

= Privacy Notices =


If you activate certain features in this plugin, the contact form submitter's personal data, including their IP address, may be sent to the service provider. Thus, confirming the provider's privacy policy is recommended. These features include:
1. reCAPTCHA ([Google](https://developers.google.com/recaptcha/));
2. Garlic.js ([Garlic](https://github.com/guillaumepotier/Garlic.js/blob/master/LICENCE.md));
3. intl-tel-input ([Phone](https://github.com/jackocnr/intl-tel-input/blob/master/LICENSE)).


= Requirements =


1. Self hosted WordPress.org installation (4.0 or higher);
2. Adlots account.


== Installation ==


1. Upload the entire Adlots Web-Form folder to the `/wp-content/plugins/` directory;
2. Activate the plugin through the 'Plugins' menu in WordPress;
3. Go to Settings->Adlots;
4. Provide Adlots account URL(required);
5. Save settings.
